This app is created in order to test different technologies.
In the map section, you can type a name of any place (address or cafe or whatever google maps can find) in the text field, this app will provide you 5 hints. Results are bounded within the border of Germany. When you choose any item from the hints provided, the google map will move to the location of that place and mark it with a marker.
This app uses:

  • Node.js
  • Angular
  • Jquery
  • Bootstrap
  • Smint Jquery plugin
  • Google Maps Javascript API
  • Font Awesome
  • The Movie Database API - Travis Bell
